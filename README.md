# Shadow Run

A 2D platformer game built with vanilla JavaScript and HTML5 Canvas. Three levels of increasing difficulty with enemy AI, collectibles, and physics-based movement.

## Play

**[Play Now] https://thesnatcher757.github.io/2D-Shadow-Run-Game/**

## Features

- 3 handcrafted levels (Dawn Valley, Crystal Caves, Sky Fortress)
- Physics-based movement with variable jump height
- Enemy AI with stomp mechanics
- Coin collecting and score tracking
- Springs, spikes, and moving platforms
- Particle effects and screen shake
- Mobile touch controls
- Lives system with invincibility frames
- Level transitions with fade effects

## Controls

- Arrow Keys or WASD to move
- Space or Up Arrow to jump
- Stomp enemies from above
- Collect coins and reach the exit

## Tech

- Zero dependencies
- Pure JavaScript and HTML5 Canvas
- Single file, no build step
- Responsive with mobile support
